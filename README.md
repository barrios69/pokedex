**PokeDex Distributed System - Angular 14**

 Descripción
Este proyecto es una aplicación web desarrollada en **Angular 14** que funciona como una PokeDex moderna. La aplicación consume datos de forma distribuida desde la **PokeAPI** utilizando protocolos REST y GraphQL (Apollo).

🚀 Tecnologías Utilizadas
* **Frontend:**: Angular 14.
* **Estado y Datos:**: Apollo GraphQL & RxJS.
* **Infraestructura:** Azure Static Web Apps.
* **CI/CD:** :GitHub Actions.
* **Seguridad:**: Content Security Policy (CSP) & HSTS.

 Instalación y Desarrollo Local
1. Clonar el repositorio: `git clone https://github.com/barrios69/pokedex.git`
2. Instalar dependencias: `npm install`
3. Ejecutar servidor local: `ng serve`
4. Abrir en: `http://localhost:4200`
